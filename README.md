# QnA - Questions and Answers

## Description
A question answer site (similair to [StackOverflow](http://stackoverflow.com) or [Quora](http://quora.com)), created using Spring technologies.

## Running the project
`mvn spring-boot:run`

## Technologies used
* Spring Web MVC (with Thymeleaf)
* Spring Boot
* Spring Security
* Spring Data JPA
